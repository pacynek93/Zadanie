Add or modify property form

  1. Below you can find all current properties listed.
  2. To manage them simply use modify or delete buttons on each list element.
  3. To add new property use 'Add new' button on the right.
  4. Be careful with 'Remove all' button, that will remove all properties and you cannot undo this action.
  5. Use 'Show all' button to reload properties list.
